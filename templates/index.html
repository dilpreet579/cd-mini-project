About MiniSQL Studio

A web-based Interactive SQL Compiler and Query Processor designed to visualize step-by-step token and AST generation pipelines.

Supported Grammar

  • CREATE TABLE tbl_name (col1, col2)
  • INSERT INTO tbl_name VALUES (val1, val2)
  • SELECT * FROM tbl_name
  • SELECT col1 FROM tbl_name WHERE col1 = val1 AND col2 = val2
  • UPDATE tbl_name SET col=val WHERE cond
  • DELETE FROM tbl_name WHERE cond
  • DROP TABLE tbl_name
